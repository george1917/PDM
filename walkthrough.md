# PDM Tool 操作演示

## 1. 系統概覽
本系統為一個基於 Streamlit 的產品資料管理 (PDM) 工具，支援單筆新增、列表檢視、Excel 匯出與批次匯入功能。
![Home Page](https://via.placeholder.com/800x400?text=Home+Page+Placeholder)

## 2. 功能操作

### 2.1 建立新產品
1. 點擊左側選單 **「建立新產品」**。
2. 填寫產品名稱、SKU (唯一識別碼)、分類、價格等資訊。
3. 可上傳產品圖片 (JPG/PNG)。
4. 點擊 **「🚀 確認建立產品」**，成功後會顯示確認訊息與氣球動畫。

### 2.2 產品列表與管理
1. 點擊左側選單 **「產品列表」**。
2. **檢視資料**: 表格呈現所有產品，包含圖片縮圖。
3. **編輯產品**:
   - 在頁面下方「✏️ 編輯產品」區域，從下拉選單選擇產品。
   - 修改欄位內容或更換圖片。
   - 點擊「💾 儲存變更」即時更新資料庫。
4. **匯出資料**:
   - 點擊表格上方的 **「📥 匯出 Excel」** 按鈕，即可下載完整產品清單 (`.xlsx`)。

### 2.3 批次上傳/更新
1. 在產品列表頁面底部的 **「📤 批次處理」** 區域，展開 **「批次上傳/更新產品 (Excel)」**。
2. 下載範本或依據以下欄位準備 Excel 檔案 (header 不分大小寫):
   - `name` (必填)
   - `sku` (必填，唯一識別)
   - `category`
   - `price`
   - `cost`
   - `description`
3. 上傳檔案後，系統會檢查欄位完整性。
4. 點擊 **「🚀 開始批次處理」**，系統將自動比對 **SKU**:
   - **若 SKU 存在**: 更新該產品資料。
   - **若 SKU 不存在**: 建立新產品。
5. 完成後顯示新增與更新的筆數統計。

## 3. 技術說明
- **Frontend**: Streamlit
- **Backend**: Python
- **Database**: SQLite (`pdm.db`)
- **Excel Engine**: Pandas + Openpyxl
